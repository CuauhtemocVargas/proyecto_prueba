¿Qué es GitHub?

GitHub es una plataforma en la que se pueden crear repositorios que contengan el código para una páginja web o para algún programa. GitHub sirve como una nube en la cual se slalva todo el código que tienes y además puedes compartirlo con más gente. Es un lugar donde puedes guardar el código que ya hgiciste, yt en caso de que algo le pase a la información en la computadora, este estará en github igualmente.
