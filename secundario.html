¿Qué es GIT?

GIT es la extensión que se usa para poder sincronizar la plataforma en la cual estas programando con GitHub. sin GIT no se podría subir el código a la nube de GitHub y por ende el código no se salvaría en la nube. en pocas palabras GIT nos sirve para poder conectar el programa de código junto con GitHub que es nuestra nube.
